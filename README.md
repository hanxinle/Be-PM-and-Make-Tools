# 自己写一些有用或者好玩的工具

## 项目1

**PC 原生闹钟** — 一只住在桌面上的硬核大猫咪，用 C++ 和 Qt 锻造而成，和浏览器里那只轻飘飘的 H5 小猫是亲兄弟，但脾气更猛。

它不搞网页那一套，直接调用 Windows 的 `PlaySound`  API 放铃声，到点了会霸气弹出一个 400x300 的 GIF 动画窗口，配上超大字号提醒你「该动身了」。32 秒后自动消失，不啰嗦、不恋战。支持添加多个闹钟，触发完就从列表里光荣退役，防止同一分钟内反复轰炸你。开会、吃饭、吃水果怕晚点？它比你还着急。

## 项目2

**硬座** — 听名字就知道不是什么正经工具，但它干的活儿可太贴心了。这也是一款 PC 原生 C++ 程序，和项目1一样走硬核路线。

它的任务很单纯：每 5 分钟偷偷帮你按一下 NumLock，让电脑以为你一直在键盘前奋笔疾书，屏幕锁？不存在的。启动后缩在一边即可，窗口里还有一只循环播放的 GIF 小动画陪你，孤单的加班夜也不至于太无聊。想下班了？点「退出程序」就行，干净利落。

## 项目3

![小猫闹钟截图](doc/cat_clock.png)

**小猫闹钟** — 一只住在浏览器里、负责叫你起床（或者起来吃饭、开会、摸鱼）的小猫咪。

它很乖，只要一个 HTML 文件就能跑，不用安装不用配环境。你可以让它同时记好多个闹钟，每个都能换上你喜欢的铃声（默认自带几首，也支持上传自己的宝藏音乐）。到点它会蹦出来一个弹窗，配上一张可爱的 GIF，赖床？不存在的。

深色渐变背景加上毛玻璃卡片，颜值在线；时间可以用滚轮搓、用键盘调、或者直接打字输入，怎么顺手怎么来。闹钟响过之后会变成「已响铃」状态乖乖待在列表里，随时查看自己今天又准时了几次。
